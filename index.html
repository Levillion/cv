Explaination of each tags

Elements within the body tag

List

  • <"ul"> unordered list - item starts with bullet point
  • <"ol"> ordered list - item starts with ordered character, use attribute to specify type of char i.e letters or numbers
Inside the list tag are <"li"> or list item tag that denotes each entries on the list.

Images

To display a image, use <"img"> with attribute src= either path to image file inside directory of the website (root is where the index.html is), or an url of a picture online.

Use alt= attribute to denote with text what this picture is about in case it cannot be displayed. The Google search engine also uses this text information to determine your website's contents for search results!

Tables

Use tag <"table"> then <"tbody">, inside use <"tr"> to create rows of the table, then <"td"> to add entries/cells in each row. Use <"thead"> to define the row as the header row, <"th"> to define one cell as a header cell.

Form

Note that the attribute of <"form">, "action"= to index.html, so when submit is pressed, the form redirects the browser to the index page. It can be set =mailto so that when submit is pressed, the system default email app will run and the email address entered will be fetched into the email app ready to compose a email. The "method= post" state that all field inputs will be transfered into email client.

The attribute enctype="text/plain" specifies that the text from all the fields are encoded in utf-8 text so that the transfered text will not contain jubberish characters like %.

The name= attributes of the inputs are the names of the variable that stores the inputed values.
